# React-фронтенд для системы морской маршрутизации

## Требования
- Node.js 16+ и npm

## Установка и запуск

1. Убедитесь, что бэкенд API запущен на `http://127.0.0.1:8000`
2. В папке `frontend-react` выполните:
   ```bash
   npm install
   npm start
```

1. Откроется браузер на `http://localhost:3000`

## Функции

- Интерактивная карта (Leaflet)
- Установка маркеров старта и финиша кликами или перетаскиванием
- Построение маршрута с выбором типа судна и оптимизации (время/безопасность)
- Отображение маршрута и информации о расстоянии
- Поиск похожих ситуаций по параметрам риска, ветра, волн

## Примечания

- Прокси настроен на `http://127.0.0.1:8000`, поэтому запросы к API не требуют указания полного URL
- Для работы требуется запущенный бэкенд (команда `python -m uvicorn app.main:app --reload` в корне проекта)
